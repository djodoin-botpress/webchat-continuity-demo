Botpress Webchat — User Continuity Demo

The webchat stores its user identity (userId + userToken + conversationId) in localStorage under bp-webchat-<clientId>. That entry is the entire identity — keep it and the same user/conversation comes back.

  1. Reload the page — the chat restores your conversation automatically.
  2. Start as new user wipes the entry and creates a fresh identity on next open.
  3. Export / Import the identity to carry the same user to another browser or device.
